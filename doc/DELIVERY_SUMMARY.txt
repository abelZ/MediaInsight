RTMP PROTOCOL SUMMARY
=====================

Handshake (3072 bytes total):
  C0: 1 byte (version 0x03)
  C1: 1536 bytes (timestamp + random)
  S0: 1 byte (version 0x03)
  S1: 1536 bytes (timestamp + random)
  S2: 1536 bytes (copy of C1)
  C2: 1536 bytes (copy of S1)

Messages:
  Type 8: Audio Data (FLV tag)
  Type 9: Video Data (FLV tag)
  Type 18: Script Data (metadata)
  Type 20: Command (Connect, CreateStream, Play)

Chunks (4 header formats):
  Format 0: Full (12 bytes) - [header | timestamp | length | type | stream_id]
  Format 1: No ID (8 bytes) - [header | delta_ts | length | type]
  Format 2: Delta (4 bytes) - [header | delta_ts]
  Format 3: Reuse (1 byte) - [header]
